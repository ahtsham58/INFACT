INFACT

Overview

In this study, we aim to evaluate chat-bot systems that provide movie recommendations based on user preferences.

Our specific goal here is to compare three chat-bots. To that purpose, we will show you dialog situations where a user interacts with the chat-bot. The shown dialogs ends at a point where the chat-bot should respond to the last sentence by the user.
Your task is to rate each of these alternatives in terms of their meaningfulness and quality.


Procedure & Instructions

  1. Please inspect the Terms and Conditions
  2. Once you start the procedure, We will then show you three possible responses by the chat-bots for the last user statement.
  3. Your task is to rate each of these responses on the given scale from “Entirely meaningless” to “Perfectly Meaningful”.
  4. Please rate each response individually; it is perfectly fine if two or all three responses receive the same rating.
  5. When is a response meaningful?
    (a)   A response should be a logical by the chat-bot, e.g., the chat-bot should make a recommendation when the user asks for one.
    (b)   If a recommendation is made by the chat-bot, it has to match the user’s stated preferences. If, for example, a user is looking for an funny movie, mostly funny movie recommendations are meaningful. Please make this assessment based on your knowledge about the mentioned movies. You may always look up services like IMDb to check, e.g., the movie genres and plots. Note that movie titles are shown in quotes " " in the dialogs.
    (c)   If the chat-bot response is not a movie recommendation, you are supposed to rate the meaningfulness of the response as a reply to the user’s last statement keeping in mind the context of the dialog situation.
  6. Once you have rated all three chat-bot responses, click on “Submit ratings” to repeat the task for the next dialog situation. We will show you 10 such dialog situations.
  7. To start the study, please confirm read the terms and conditions using the checkbox and click on “Start Procedure”.

    8. Please read point 5 again.
    There you go...
